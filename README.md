# Gulf Star Hotel - Digital Transformation

This repository contains the finalized, high-end luxury hospitality website for the **Gulf Star Hotel** in Hargeisa.

## Features
- **Cinematic Video Hero:** Immersive auto-playing video background.
- **Interactive Booking Engine:** High-conversion booking bar for guest inquiries.
- **Curated Experiences:** Professional gallery with brand-pattern background.
- **Interactive Map:** Custom-styled Google Maps integration.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **WhatsApp Integration:** Floating chat button for direct guest communication.

## Deployment to Vercel

1. **Upload to GitHub:**
   - Create a new repository on GitHub.
   - Push all files from this folder to the repository.

2. **Connect to Vercel:**
   - Log in to [Vercel](https://vercel.com).
   - Click "Add New" > "Project".
   - Import your GitHub repository.
   - Click **Deploy**.

3. **Final Polish:**
   - Vercel will automatically detect the `vercel.json` and host your static files with optimized headers.

## Directory Structure
- `/assets`: Contains all images, logos, and the cinematic hero video.
- `/css`: Contains `styles.css` (The core luxury design system).
- `/js`: Contains `main.js` (Interactivity and scroll logic).
- `index.html`: The main entry point.
- `vercel.json`: Deployment configuration.
